# RetroPie Gaming Kit

Adding value to your life

A ready-to-play retro console based on **RetroPie**. *(No ROMs included.)* **Docs & files:** - GitHub repo → - Quickstart (HTML) → `retropie/docs/quickstart.html` in the repo - FAQ → ## What’s in the kit - Raspberry Pi 5 (4 GB) with fan case - 64 GB A1 microSD (RetroPie pre-flashed) - 2× wireless controllers · PSU · Micro-HDMI cable ## First boot 1. Insert SD, connect HDMI and power. 2. Pair controllers in EmulationStation → “Configure Input”. 3. Add your own ROMs via network share `\\retropie\roms\` or USB.
